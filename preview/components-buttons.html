Component Reference

Button
Components

Five button variants for the Bootlegger brand — primary, secondary, gold, ghost, and outline. Each built with --text-button (0.9375rem / 500 weight / sans-serif) on --radius-md (8px) with 0.75rem/1rem vertical/horizontal padding.

Token Reference

Button Anatomy

Every button variant shares a base class and then applies variant-specific background, color, and border tokens. The base shape is always the same.
